《ES6标准入门》学习笔记第3章 变量的解构赋值

2019-06-04

基本概念

ES6允许按照一定模式从数组和对象中提取值，然后对变量进行赋值，这被称为解构。

解构赋值的规则是，只要等号右边的值不是对象或者数组，就将其转换为对象，undefined和null无法转换为对象，所以会报错

3.1 数组的解构赋值

本质上这种写法属于“模式匹配”，只要等号两边相同，左边的变量就会被赋予对应的值。

如果解构不成功，变量的值就等于undefined，如果等号右边不是数组，（严格意义上不具有iterator接口）就会报错。

还允许对解构变量使用默认值

3.2 对象的解构赋值

对象的解构与数组有所区别，数组的解构是由元素的位置决定的，而对象的解构要求变量必须与属性同名才能取到正确的值
