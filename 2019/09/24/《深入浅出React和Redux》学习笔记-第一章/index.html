新建一个React组件

使用create-react-app构建一个react项目

create-react-app是react官方开源的一个react项目脚手架，通过npm安装它

npm i creat-react-app -g

使用create-react-app创建一个新的react项目
create-react-app first_react_app

启动本地调试环境
npm start

create-react-app的配置文件都放在了node_modules/react-scripts目录下，但我们可以通过eject命令去暴露配置文件，此过程不可逆。
npm run eject

使用create-react-app构建一个react项目

首先要在脚本的开头引入Component和React

Component是所有组建的基类，所有的组件都要继承这个基类，在有JSX的地方就要引入React

JSX，JavaScript的语法扩展，可以让我们在JS中编写HTML代码，React组件必须要首字母大写，

UI = render(data)
React是数据驱动渲染的，页面的变化由数据决定

React 组件的数据

React组件的渲染由数据驱动，数据分为两种，prop和state，简单地理解，prop是对外的接口，state是内部的状态。

React的prop

在React中，prop（property）是聪外部传递给组件的数据，组件可以通过定义自己的prop来决定自己对外接收数据的接口
