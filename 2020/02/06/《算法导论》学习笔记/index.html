动态规划

动态规划（dynamic programming）与分治法相似，都是通过组合子问题来求解原问题，在这里programming指的是一种表格法。动态规划应用于子问题重叠的情况，即不同的子问题具有公共的子子问题。
在这种问题的情况下，分治法会做许多不必要的工作，它会重复求解那些公共子问题，而动态规划对子问题只求解一次，然后将解保存在表格中，从而避免重复的计算工作

动态规划通常用来求解最优化问题。通常使用以下四个步骤来设计动态规划算法

  1. 刻画一个最优解的结构特征
  2. 递归地定义最优解的值
  3. 计算最优解的值，通常采用自底向上的方法
  4. 利用计算出的信息构造一个最优解

锯钢条问题

给定一段长度为n的钢铁和一个价格表pi(i=1, 2, 3,…,n)，求切割钢条的方案，使得收益最大，钢条可以不切割。

长度为n的钢条共有2^(n-1)种不同的切割方案，因为在距离钢条左端i（i=1, 2, 3, …, n-1）处，我们总是可以选择切割或者不切割

常规递归法

将钢条从左边切下长度为i的一段，然后对右边剩下的长度n-i继续进行切割，进行递归求解
rn = max(pi + rn-1), 1<=i<=n
